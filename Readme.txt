Prediction of wine Recommendation.

This project will utilize machine learning models, and determine if the effect of country where the wine was produced, and the sell price of wine affects rating prediction.

Zip file, wine_review_data.csv.zip, contains the dataset, and must be downloaded and reside in your working directory before executing Rmd or R script.

The three files are:
1 - Project_wine.pdf – Project report in pdf format
2 - Project_wine.Rmd – Rmd file used for generating report 
3 - Project_wine.R   – R script file
